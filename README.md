 SkinGenie 

SkinGenie is an AI-based skin analysis project that classifies basic skin conditions using machine learning.

 Features
- Skin type classification
- Image-based prediction
- Simple and fast implementation

 Tech Stack
- Python
- TensorFlow / Scikit-learn
- OpenCV

Project Structure
- main.py → Main code
- model.py → ML model
- dataset → Images (optional)

How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python main.py
